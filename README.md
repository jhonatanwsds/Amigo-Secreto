## 🚀 Funcionalidades

- **Modo Clássico:** Sorteio tradicional de amigos secretos  
- Interface totalmente temática do Spy x Family  
- Personagens interativos (Loid, Yor e Anya)  
- Sistema de sons e animações

- Exportação de resultados em texto
